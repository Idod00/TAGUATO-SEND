Ayuda y Guia de Uso

Bienvenido a TAGUATO-SEND. Aqui encontraras una guia paso a paso para cada funcion del portal. Haz clic en cada tema para expandirlo.

Una instancia representa una conexion a un numero de WhatsApp. Cada numero requiere su propia instancia.

Crear una instancia WhatsApp

  1. Ve a la seccion Instancias en el menu lateral.
  2. Escribe un nombre para la instancia (solo letras, numeros y guiones). Ej: ventas-principal
  3. Selecciona WhatsApp en el selector de canal.
  4. Haz clic en Crear.

Conectar a WhatsApp (escanear QR)

  1. Una vez creada la instancia, aparecera en la lista con estado Desconectado.
  2. Haz clic en el boton Conectar / QR de la instancia.
  3. Se abrira un modal con el codigo QR. Escanea con WhatsApp en tu telefono:
    WhatsApp → Dispositivos vinculados → Vincular dispositivo
  4. Espera unos segundos. El estado cambiara a Conectado.
El QR expira en ~60 segundos. Si vence, haz clic de nuevo en Conectar para obtener uno nuevo.

Estados de la instancia

  • Conectado — Listo para enviar mensajes.
  • Conectando — Sincronizando con WhatsApp.
  • Desconectado — Requiere reconexion.
Si cierras sesion en el telefono, la instancia se desconectara y deberas escanear el QR nuevamente.

Las instancias Telegram funcionan mediante un Bot de Telegram. No requieren escanear QR ni mantener un telefono conectado.

Paso 1: Crear el bot en Telegram

  1. Abre Telegram y busca @BotFather.
  2. Envia el comando /newbot.
  3. BotFather te pedira un nombre para el bot (ej: Mi Bot de Ventas) y luego un nombre de usuario que debe terminar en bot (ej: miventas_bot).
  4. BotFather te entregara un token con formato: 1234567890:AAHb...
  5. Guarda ese token, lo necesitaras en el siguiente paso.

Paso 2: Crear la instancia en el portal

  1. Ve a Instancias → escribe el nombre de la instancia.
  2. Selecciona Telegram en el selector de canal.
  3. Pega el token que te dio BotFather en el campo Bot token.
  4. Haz clic en Crear. La instancia quedara activa de inmediato.
Los bots de Telegram no se desconectan como WhatsApp. El sistema no intentara reconectarlos automaticamente.
El bot solo recibe mensajes de usuarios que le hayan enviado /start previamente. Para enviar mensajes debes conocer el Chat ID del destinatario.

Desde la seccion Mensajes puedes enviar mensajes individuales de texto o con archivos adjuntos.

Mensaje de texto

  1. Selecciona la instancia que usaras para enviar.
  2. Ingresa el numero de destino con codigo de pais, sin el signo +. Ej: 50688888888
  3. Escribe el mensaje en el campo de texto.
  4. Haz clic en Enviar.

Mensaje con archivo (imagen, video, audio, documento)

  1. Selecciona el tipo de archivo en el selector de tipo de mensaje.
  2. Pega la URL publica del archivo o sube el archivo directamente.
  3. Opcionalmente agrega un texto como descripcion.
  4. Haz clic en Enviar.
Puedes usar emojis haciendo clic en el icono de cara sonriente junto al campo de texto.

El envio masivo te permite enviar el mismo mensaje a multiples numeros en una sola operacion.

  1. En la seccion Mensajes, desplazate a la parte de Envio Masivo.
  2. Selecciona la instancia de envio.
  3. Ingresa los numeros destinatarios separados por coma o en lineas separadas. Ej:
    50688888888, 50677777777
  4. También puedes cargar una lista de contactos guardada haciendo clic en Cargar Lista.
  5. Escribe el mensaje y haz clic en Enviar a Todos.
El envio se realiza con un intervalo entre mensajes para evitar bloqueos. Puedes cancelar en cualquier momento con el boton Cancelar.
Enviar mensajes no solicitados puede resultar en el bloqueo del numero por parte de WhatsApp. Usa esta funcion responsablemente.

Las plantillas son mensajes predefinidos que puedes reutilizar para ahorrar tiempo.

Crear una plantilla

  1. Ve a la seccion Plantillas.
  2. Escribe un nombre descriptivo (ej: Bienvenida).
  3. Escribe el contenido del mensaje.
  4. Haz clic en Crear.

Usar una plantilla al enviar

  1. En el formulario de envio de mensajes, haz clic en Usar Plantilla.
  2. Selecciona la plantilla deseada. El contenido se cargara automaticamente.
  3. Modifica el texto si es necesario y envia.

Organiza numeros de telefono en listas de contactos para usarlos en envios masivos.

Crear una lista

  1. Ve a la seccion Contactos.
  2. Escribe el nombre de la lista y haz clic en Crear Lista.
  3. Haz clic en la lista recien creada para abrirla.
  4. Agrega contactos: ingresa el numero (con codigo de pais) y opcionalmente una etiqueta.

Usar una lista en mensajes masivos

  1. En el formulario de envio masivo, haz clic en Cargar Lista.
  2. Selecciona la lista deseada. Los numeros se cargaran automaticamente.
Puedes tener multiples listas para diferentes segmentos: clientes, proveedores, equipo interno, etc.

El historial registra todos los mensajes enviados desde el portal.

  • Filtra por instancia, numero, estado o rango de fechas.
  • Cada registro muestra instancia, numero, tipo de mensaje, estado y fecha.
  • Estados: Enviado Fallido Cancelado

Exportar a CSV

  1. Aplica los filtros deseados.
  2. Haz clic en Exportar CSV para descargar el historial filtrado.

Programa mensajes para que se envien automaticamente en una fecha y hora especificas.

  1. Ve a la seccion Programados.
  2. Selecciona la instancia y los destinatarios (o carga una lista).
  3. Escribe el mensaje.
  4. Elige la fecha y hora de envio.
  5. Haz clic en Programar Envio.
El sistema revisa los mensajes pendientes cada minuto. El envio puede ocurrir con hasta ~60 segundos de retraso respecto a la hora exacta.
La instancia seleccionada debe estar conectada al momento del envio. Si esta desconectada, el mensaje quedara como fallido.

Los webhooks permiten que tu servidor reciba notificaciones en tiempo real cuando ocurren eventos en tus instancias (mensajes recibidos, cambios de estado, etc.).

Configurar un webhook

  1. Ve a la seccion Webhooks.
  2. Selecciona la instancia.
  3. Ingresa la URL de tu servidor que recibira los eventos (debe ser HTTPS y publica).
  4. Selecciona los eventos que quieres recibir.
  5. Haz clic en Crear Webhook.
La URL del webhook debe ser accesible desde internet. No puedes usar localhost ni IPs privadas.
Si el webhook falla repetidamente, el sistema lo marcara como inactivo. Revisa que tu servidor responda con HTTP 200.

La seccion Sesiones muestra todos los dispositivos y navegadores donde tienes sesion activa.

  • Cada sesion muestra IP, navegador y ultima actividad.
  • Puedes cerrar sesiones individuales sospechosas.
  • El boton Cerrar todas las sesiones invalida todos los tokens activos, incluyendo el actual.
Las sesiones expiran automaticamente despues de 24 horas de inactividad.

TAGUATO-SEND expone una API REST completa para que puedas integrar el envio de mensajes en tus propias aplicaciones.

Autenticacion

Todas las solicitudes requieren el header apikey con tu token de sesion:

apikey: tu-token-de-sesion

Obtén tu token haciendo login en POST /api/auth/login.

Endpoints principales

  • POST /api/auth/login — Iniciar sesion
  • GET /instance/fetchInstances — Listar instancias
  • POST /instance/create — Crear instancia
  • POST /message/sendText/{instancia} — Enviar texto
  • POST /message/sendMedia/{instancia} — Enviar archivo
Consulta la seccion API Docs en el menu para ver la referencia completa con todos los parametros y ejemplos.

Cambiar contrasena

  1. Haz clic en tu nombre de usuario en la parte superior del menu lateral.
  2. Selecciona Cambiar contrasena.
  3. Ingresa tu contrasena actual y la nueva contrasena.
  4. La nueva contrasena debe tener minimo 8 caracteres, una mayuscula, una minuscula y un numero.
Al cambiar la contrasena, todas tus sesiones activas seran cerradas y deberas iniciar sesion nuevamente.

Bloqueo por intentos fallidos

Despues de 5 intentos de login fallidos, la cuenta se bloquea por 15 minutos automaticamente.
